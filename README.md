# DrawSea
タブレット端末からお絵描きした魚をスクリーンに送信し、泳がせるためのアプリです。
> ページへ以下のURLよりアクセスできます。
> https://mecanam.github.io/DrawSea/

## Usage

### PC（スクリーン側）
1. 上記URLにアクセスするとルームIDとQRコードが表示されます
2. 魚の寿命（デフォルト5分）を必要に応じて設定します
3. 「起動する」ボタンを押すとフルスクリーンの水族館画面に切り替わります
4. マウスを動かすと右上にメニューが表示され、リセットや終了が行えます

### タブレット（お絵かき側）
1. QRコードをスキャン、またはルームIDを手入力して接続します
2. ペン・消しゴム・太さ・カラーパレットを使って魚を描きます
3. 魚の向き（右向き/左向き）を選択します
4. 「おくる」ボタンを押すとスクリーンに魚が泳ぎ出します

## 技術構成
- **通信**: PeerJS（WebRTC P2P） - サーバーレスで動作
- **音声**: Web Audio APIによる環境音・効果音生成
- **ホスティング**: GitHub Pages（静的サイトのみ）
